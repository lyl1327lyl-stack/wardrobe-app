**日期：2026-04-06**

## 一、搭配详情页布局

### 1.1 整体结构

```
┌─────────────────────────────────┐
│  ←  搭配名称(可点)       操作栏 │  ← 顶栏
├─────────────────────────────────┤
│                                 │
│      [画板区域]                 │  ← 可拖动/缩放衣物
│   自由拖动排列搭配效果          │
│                                 │
├─────────────────────────────────┤
│  保存  │  改名  │  删除         │  ← 底部操作栏
├─────────────────────────────────┤
│  从下方拖动或点击添加衣物       │  ← 底部Strip
│  [衣1][衣2][衣3]...            │
└─────────────────────────────────┘
```

### 1.2 底部操作栏

- **保存**：保存当前画板布局和衣物位置
- **改名**：点击弹出输入框直接编辑名称
- **删除**：删除整个搭配，需二次确认

### 1.3 画板交互

| 动作 | 效果 |
|------|------|
| 点击衣物 | 选中该衣物，显示删除按钮(右上角×) |
| 拖动衣物 | 移动到任意位置，松手后保存位置 |
| 再次点击其他区域 | 取消选中状态 |

---

## 二、衣物删除交互

### 选中后显示删除按钮

- 点击画板上的衣物 → 该衣物显示红色删除按钮(×)
- 点击删除按钮 → 弹出确认框"确定要从该搭配移除这件衣服？"
- 确认后移除，页面刷新Strip显示被移除的衣物

---

## 三、添加衣物交互

### 拖动 + 点击双模式

**拖动添加**：
- 从底部Strip拖动衣物图片到画板区域
- 衣物出现在松开位置
- 可立即拖动调整

**点击添加**：
- 点击底部Strip中的衣物，自动添加到画板中央
- 位置固定在中央（SCREEN_WIDTH/2 - 60, CANVAS_HEIGHT/2 - 60）

## 四、搭配列表缩略图

### 忠实呈现画板布局

- 直接将画板内容截取为缩略图作为搭配封面
- 保持画板宽高比（1:1.2）
- 画板布局完全忠实呈现

## 六、添加衣物弹层

### 点击加号 → 弹层多选

**交互流程**：
1. 底部Strip右侧显示 "+" 图标按钮
2. 点击 "+" 弹出 Modal
3. 顶部搜索框 + 两组横向分类导航（类型 + 季节）
4. 下方为衣物网格，点击切换选中状态
5. 底部显示已选数量和「确认添加」按钮
6. 确认后批量添加衣物到画板

**Modal 布局**：
```
┌──────────────────────────────────────┐
│  选择衣物                      [×]   │
├──────────────────────────────────────┤
│  🔍 搜索品牌/场合/风格/颜色/备注    │
├──────────────────────────────────────┤
│  类型                               │
│  [全部][上衣][裤子][裙子][鞋子][配件][外套] │
│  季节                               │
│  [全部][🌸春][☀️夏][🍂秋][❄️冬]      │
├──────────────────────────────────────┤
│  [衣1✓][衣2][衣3][衣4][衣5][衣6]... │
│                                      │
│                                      │
├──────────────────────────────────────┤
│  已选 3 件              [确认添加]   │
└──────────────────────────────────────┘
```

**筛选逻辑**：
- 类型筛选：全部/上衣/裤子/裙子/鞋子/配饰/外套
- 季节筛选：全部/春/夏/秋/冬
- 类型和季节同时生效，取交集
- 搜索关键词匹配：品牌、场合、风格、颜色、备注
- 搜索与分类同时生效

## 七、实现优先级

1. **高**：底部操作栏 + 改名/删除逻辑
2. **高**：删除确认框
3. **高**：添加衣物弹层（分类导航 + 多选）
4. **中**：拖动添加衣物到画板（保留点击添加作为备选）
5. **低**：画板缩略图生成

## 八、技术要点

### 拖动添加实现
- 使用 PanResponder 监听 Strip 物品的拖动
- 拖出 Strip 区域时，显示画板"可放置"提示
- 释放时计算相对于画板的位置，添加到 localItemIds/localPositions

### 画板缩略图
- 方案A：使用 react-native-view-shot 截图
- 方案B：Canvas 组件渲染（需安装 react-native-skia）
- 推荐方案A，实现更简单

### 状态管理
- localItemIds: number[] — 待保存的衣物ID列表
- localPositions: Record<number, OutfitItemPosition> — 位置信息
- hasUnsavedChanges — 对比原始数据判断是否有修改
